# The 8086mc Project

Implementation of simple 8086-like computers in Minecraft

## Building

To build for Fabric 1.21.10:

```sh
./gradlew build
```

You can download the latest development artifact from [nightly.link](https://nightly.link/GIKExe/The-8086mc-Project/workflows/build/main/Artifacts.zip)
